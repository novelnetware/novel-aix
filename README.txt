An intelligent, customizable AI chatbot for WordPress, supporting OpenAI, Gemini, and Deepseek models. Modern UI, content-aware, and easy to set up.

== Description ==
Novel AI Chatbot is a powerful, modern chatbot plugin for WordPress, powered by leading AI models (OpenAI, Gemini, Deepseek). It can answer questions about your website, summarize your content, and provide a beautiful, customizable chat experience for your visitors.

* Supports OpenAI, Gemini, and Deepseek APIs
* Collects and summarizes your site content for context-aware answers
* Modern, fully customizable chat widget (colors, icons, logos, welcome message, etc.)
* Floating or shortcode-embedded display
* Admin live preview and easy SVG/PNG uploads
* Caches and batches content collection for API efficiency
* Never loads in the WordPress admin area
* "Powered By Novelnetware" branding with support link

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/novel-ai-chatbot` directory, or install via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Novel AI" in your WordPress admin menu to configure settings, API keys, and customize the chat widget.
4. (Optional) Use the `[novel_ai_chatbot]` shortcode to embed the chatbox in posts or pages.

== Frequently Asked Questions ==
= Which AI models are supported? =
OpenAI (ChatGPT), Google Gemini, and Deepseek.

= How do I get API keys? =
Links to each provider are in the plugin settings. You must sign up and generate your own API keys.

= Can I customize the chat widget? =
Yes! You can change colors, icons, logos, welcome messages, and more from the admin panel.

= Does the chatbot use my site content? =
Yes, it collects and summarizes your posts/pages (and soon sitemap URLs) to provide context-aware answers.

= Is my data secure? =
Your API keys are stored securely in your WordPress database. Only admins can access them.
